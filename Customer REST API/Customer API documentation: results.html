Customer REST API Method: results

 

Retrieves the complete collection of results for a particular test and version.

 

URL:

https://#{company}.litmus.com/tests/#{test-id}/versions/#{version}/results.xml

 

HTTP Method(s):

GET

 

Parameters:

  • test-id.  Required. The ID of the test you wish to retrieve

  • version. Required. The version of this test you wish to retrieve. 



 


Response:
