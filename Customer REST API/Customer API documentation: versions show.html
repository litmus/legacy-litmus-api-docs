Customer REST API Method: versions/show

Returns details of a single version for a particular test.

 

URL:

https://#{company}.litmus.com/tests/#{test-id}/versions/#{version}.xml

 

HTTP Method(s):

GET

 

Parameters:

  • test-id.  Required. The ID of the test you wish to retrieve

  • version. Required. The version of this test you wish to retrieve.
