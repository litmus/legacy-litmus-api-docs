Customer REST API Method: tests/show

Returns details of a single test.

 

URL:

https://#{company}.litmus.com/tests/#{id}.xml

 

HTTP Method(s):

GET

 

Parameters:

  • id.  Required. The ID of the test you wish to retrieve
