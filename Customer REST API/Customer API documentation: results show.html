Customer REST API Method: results/show

 

Used to retrieve details of a single result, useful when used in conjunction with the versions/poll method while waiting for individual results to complete.

 

URL:

https://#{company}.litmus.com/tests/#{test-id}/versions/#{version}/results/#{result-id}.xml

 

HTTP Method(s):

GET

 

Parameters:

  • test-id.  Required. The ID of the test you wish to retrieve
  • version. Required. The version of this test you wish to retrieve.
  • result-id. Required. The ID of the result you wish to retrieve.


 


Response:
