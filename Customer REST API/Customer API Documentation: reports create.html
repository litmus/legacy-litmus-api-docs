Customer REST API Method: reports/create

 

Creates a new report if you have available credits.

 

URL:

https://#{company}.litmus.com/reports.xml

 

HTTP Method(s):

POST

 

Parameters:

  • n/a 

 

Request: 

          XML example: 

 

<?xml version="1.0" encoding="UTF-8"?>
<report>
  <name>Test name</name>
</report>

Response: 

       XML example:
