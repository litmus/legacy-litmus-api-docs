Reseller API 

The reseller API can only be used if you have spoken to the Litmus sales team and signed a reseller agreement. If you just have a Litmus account then you need to use the Customer API below.

Reseller API documentation

 

Litmus Customer API

The customer API provides programatic access to your account when you have a subscription to Litmusapp.com. You cannot use this API to resell the service and it differs from the reseller API substantially. Before you start any integration using this API you should check this is the right API for you to use!
